Changelog

Version 2024.10.04a_RC3 (service release)

  • Small improvements, mostly under-the-hood.

Version 2025.09.14a_RC2

  • Added zoom panel with fixed zoom increments.
  • Zoom via mouse-wheel, + and - keys, select zoom level from pop-up, or key in zoom level manually.
  • Help has been updated.
  • Bug-fix: fixed a bug with dragging dithered images out of Ditherista.
  • Bug-fix: custom quantified palette will be re-calculated when picture brightness, gamma, contrast, etc. change,
  • Bug-fix: dot diffusion memory access error fixed.
  • Qt version support has been updated (6.9.2)

Version 2025.07.14a_RC1

  • Color dithering has been added:
    • Support for error diffusion and ordered color dithering.
    • Choose from 3 different color reduction methods.
    • Choose from 10 different color distance measurement methods.
    • Choose from multiple built-in color palettes from various computer systems.
    • Supports loading Paint.NET palettes, such as available at https://lospec.com
  • Other colors than black and white can now be chosen for mono dithering.
  • Dithering of images with transparency is now supported.
  • You can now zoom the picture in ('+' and '=' keys) and out ('-' key) or use the mouse wheel to zoom. Double clicking on the image, or pressing '.' resets the zoom.
  • Added numerical inputs for gamma, brightness and contrast.
  • Qt version support has been updated (6.9.1)

Version 2023.05.12b

  • Bugfix version for macOS Ventura and newer.
  • Fixes repeated permission pop-ups on macOS when using the file save dialog.

Version 2023.05.12a

  • First release
