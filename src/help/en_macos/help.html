GUI Overview


Loading an Image

Images can be opened in 3 ways:
  • Via the File > Open... function in the menu bar.
  • Drag and drop a supported image into the Ditherista window.
  • Copy an image from the clipboard by pressing Command + V or choosing Edit > Paste from the menu.

Dithering an Image

Choose a ditherer from the dither list on the right. Once dithered, Ditherista will cache the dithered image, so you can return to previous dithers without re-calculation. The cache will be reset when the input image settings change. You can also use the cursor keys to move from one ditherer to the next.
The colored icon left to each ditherer shows what underlying algorithm a ditherer uses.
Use the tabs at the top to choose between Mono and Color dithering.

Savinging an Image

The current dithered image can be saved in 3 ways:
  • Via the File > Save and File > Save As... functions in the menu bar.
  • Drag the image out of the Ditherista viewport into a folder in the Finder.
  • Copy the image to the clipboard by pressing Command + C or choosing Edit > Copy from the menu. Then paste the image into the target application.

Input Image Adjustments

Images can be further using the following controls:
  • Brightness
  • Contrast
  • Gamma level
  • Saturation (for Color dithering only)
Press the X button next to each control to reset the slider to its default value.

Use the show original image toggle to preview the adjustments on the original image. This is preferable when using ditherers that take long to compute.

Note: internally, Ditherista performs all calculations in linear color space for higher accuracy.

Zoom

You can zoom in and out via:
  • The + and - keys.
  • By scrolling the mouse wheel.
  • By typing a percentage into the zoom level widget at the bottom status bar.
  • By choosing a pre-defined zoom level from the zoom level widget at the bottom status bar.
Note: going below 100% zoom can be useful on a HiDPI screen.
