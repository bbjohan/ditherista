GUI Overview


Loading an Image

Images can be opened in 3 ways:

  • Via the File -> Open... function in the menu bar.
  • Drag and drop a supported image into the Ditherista window.
  • Copy an image from the clipboard by pressing CTRL+V or choosing Edit -> Paste from the menu.

Dithering an Image

Choose a ditherer from the dither list on the right. Once dithered, Ditherista will cache the dithered image, so you can return to previous dithers without re-calculation. The cache will be reset when the input image settings change. You can also use the cursor keys to move from one ditherer to the next.
The colored icon left to each ditherer shows what underlying algorithm a diterer uses.

Savinging an Image

The current dithered image can be saved in 3 ways:
  • Via the File -> Save and File -> Save As... functions in the menu bar.
  • Drag the image out of the Ditherista viewport into your operating system's file manager.
  • Copy the image to the clipboard by pressing CTRL+C or choosing Edit -> Copy from the menu. Then paste the image into the target application.

Input Image Adjustments

Images are automatically adjusted when loaded - i.e. Ditherista will convert them from the sRGB color space to the linear color space.

Images can be further adjusted using the following controls:

  • Brightness
  • Contrast
  • Gamma level
Press the X button next to each control to reset the slider to its default value.

Use the "show original image" toggle to preview the adjustments on the original image. This is preferrable when using ditherers that take long to compute.

Ditherers

Allebach Dithering

Use the "Randomize" option to introduce some randomization into the dithering, to make it look less uniform.

DBS Dithering (Direct Binary Search)

The DBS ditherer is the slowest dithering algorithm. Choose from 7 different formulas for generating the underlying dithering matrix. Formulas with higher numbers will result in more coarse dithers.

Grid Dithering

Grid dithering processes the image in pixel-blocks of a certain width and a certain height, which can be adjusted. "Min Pixels" acts as a threshold. The higher it is set, the brighter a grid block will be. Use the "Alternate Algorithm" toggle to switch between 2 different dithering formulas.
